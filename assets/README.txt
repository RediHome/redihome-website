RENDERINGS / IMAGES GO HERE
===========================

This folder holds the photos and renderings shown on the site. Until you add
them, every image slot shows a clean line-art house placeholder, so the site
looks finished even with this folder empty.

HOW TO ADD A RENDERING
----------------------
1. Drop the image file in this folder, e.g.:
      assets/tuggle-front.jpg
      assets/tuggle-rear.jpg
      assets/tuggle-floorplan.jpg
2. Point the site at it:
   • LANDING PAGE (index.html): find the PLANS array near the bottom, set the
     plan's   img:""   to   img:"assets/tuggle-front.jpg"
   • PLAN VIEWER (plans.html): find the PLANS array, set the matching view's
     img:""   to   img:"assets/tuggle-front.jpg"   (one per view: Front/Rear/Floor Plan)
3. Save, commit, push. Done.

RECOMMENDED FORMAT
------------------
JPG or PNG, ~1600px wide, landscape for elevations. Keep files under ~500 KB
each so the page stays fast.
